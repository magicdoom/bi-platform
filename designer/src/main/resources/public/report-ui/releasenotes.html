DataInsight Release Notes

Release Notes - DataInsight - Version 0.5.1 (May 10, 2013)

Important upgrade note

运营产品：盘古-商机池报表模块接入平台服务。

Highlights

  • 表格展现：单维度下钻、面包屑导航、自定义列格式；
  • 图形展现：定制化烧杯图；
  • 报表制作：拖拽生成表格、拖拽生成折线图；线下配置复杂布局的报表；
  • 平台整体：Token式访问控制；Web Session无关的集群服务模式；

---

Release Notes - DataInsight - Version 0.7.0.RC (Apr 19, 2013)

Important upgrade note

运营产品：聚宝盆-ACRM模块接入平台服务。
* 使用Dirty Solution兼容0.5.1版本，会在正式版发布时解决。

Highlights

  • 表格展现：支持多维度嵌套表、维度下钻及展开、表格排序等动作；
  • 报表参数：输入框（简单、带提示）、单选下拉框、复杂日期类、单选树型等交互控件；
  • 报表制作：拖拽生成表格、拖拽生成折线图；线下配置复杂布局的报表；
  • 平台整体：提供Standlone、Integration两种模式；

New Feature

  • 下载功能：图形、表格数据以csv格式下载；
  • 图形展现：富交互型趋势图展现；

--- ---

Known limitations

  • 表格展现：暂不支持条件格式、计算列功能
  • 报表制作：尚未整体线上支持简单布局（上图下表，图表联动）类报表制作
  • 平台整体：缺少演示环境、平台管理模块功能薄弱
